## 前提

结构的根本动作是自我维持——让消亡拉长。

自我维持有多种方式：稳定型（晶体、稳定循环）、收缩型（低耗、休眠）、周期型（振荡、周期）、膨胀型（生命体、企业、帝国）。

**这篇讲的是膨胀型维持的那一支。**因为只有它会向外伸、碰到别的结构——也只有它会进入后面这整套碰撞/依靠/审视/坍塌的循环。稳定、收缩、周期型的结构——各自维持自己，不向外伸手——不走这个循环。

在膨胀型维持这一支里——

膨胀需要碰撞。没有碰撞 → 没有"外" → 没有边界 → 结构不成立。

**膨胀不是目的，碰撞不是意外——都是这类结构维持自身的方式。**

---

## 循环

### 1. 膨胀 → 碰撞 → 彼此出现 → 依靠

在膨胀型维持这里——向外伸是默认动作。

伸势必触及某处——那一触及就是碰撞。碰撞不是偶然事件——是膨胀的必然面。

碰撞发生的那一下——**"彼此"才出现**。

不是先有两个结构、然后相遇——是膨胀碰到"不是自己的什么"的那一瞬，"自己"和"对方"被这一碰同时划出来。

"两"不在碰撞之前就有——"两"是碰撞的副产品。

一有"两"——就有"跨"——就已经是依靠的雏形。

依靠不是碰撞之后的行为——是碰撞这件事本身的另一个面。对方的存在本身就在确认你——不需要你独自去膨胀。依靠替代了独自膨胀，但依靠本身仍然是膨胀——只是借对方来胀，而不是自己胀。省力。

### 2. 依靠 → 偏好 → 追求更多交互

依靠满足了维持需求。第二次、第三次碰撞时——自我有了对照：以前自己胀耗力、现在借对方胀省力——这个对照出来的那一下，就是偏好。

偏好不是第一次就有的——它需要"这次 vs 以前"的对照——即时的碰撞里没这个对照。

偏好 → 想要更多依靠 → 需要更多交互。

### 3. 交互加深 → 了解对方的结构

交互点上，双方互相了解。了解不是目的——是追求更多依靠的副产品。自我不是想了解对方，是想靠得更近，了解是靠近过程中不可避免的附带。

### 4. 了解对方 → 必然审视自身

你没法"只看对方"——看到对方怎么运作的那一刻，就已经在拿自己做对照。理解本身就是对照。

### 5. 审视被自我收编

审视在自我内部被框定为特定角度——自我保护，只从"安全"的方向看自己。

同时，审视本身被收编为自我的新结构——"我是一个会审视自己的人"——自我不但没被拆掉，反而更厚了。

## 依靠与依赖

不是两个东西——是同一个东西的不同程度。连续光谱，没有临界点。

- 轻度：对方在外面，我借用它。拿走了，我还是我，只是累一点。
- 深度：对方被编进了内部结构。拿走了，自我直接受损——不是外部损失，是内部坍塌。

转变在交互加深的过程中连续发生。


### 6. 对方的结构进入内部 → 交汇与碰撞

对方的结构带着对方的角度进来。这个角度不受自我控制。

- **交汇：** 对方的角度与自我审视角度一致 → 确认，依靠加深。
- **碰撞：** 对方的角度与自我审视角度有差异 → 自我从未设防的方向被看见 → 坍塌危险。

坍塌不是裂缝。裂缝是局部的，坍塌是结构性的——自我用来维持自己的叙事，被从根部动摇。

### 7. 面对坍塌危险 → 三条路

- **排斥（最常见）：** 自我还够强，能承受失去依靠的代价 → 推开对方。成本最低，不用改自己，不用塌。
- **重新框定：** 自我够灵活，能把对方的角度也收编进来 → 关系继续，但自我又厚了一层。
- **坍塌：** 自我已经被动摇到无法维持 → 真的塌了一部分。

### 8. 回到膨胀

无论排斥还是坍塌，只要自我未消亡，都回到同一个地方——膨胀。膨胀是这类结构的默认动作，是它维持自身的方式。

坍塌后：虚无暴露 → 自我不可忍受 → 重新创造存在来填上 → 新结构长出来（往往比原来更厚——"我不会再让这件事发生"本身就是新的一层壳）。

排斥后：失去依靠 → 回到自己膨胀的老路。

### 9. 膨胀 → 长大或碰撞

膨胀的两个结果：

- **长大：** 膨胀暂时没碰到新结构，自我变大。
- **碰撞：** 膨胀过程中遇到新的结构 → 回到第1步。

长大只是暂时的状态——体积变大就是碰撞面变大——下一次碰撞只是早晚的事。

**循环没有出口。只要自我在，就在这个循环里。**

## 关于这整套描述

循环成立——是因为"结构"、"自我"、"依靠"、"碰撞"都已经被默认存在了。

一旦默认——"必定如何如何"就长出来——这套循环就是长出来的东西。

这不是说循环是假的。结构运作时——这就是结构的样子。

但这整套描述——也是显现的一次。

不是"看穿它就出循环"——是描述本身和循环一样，都是这一下的显现。

循环在运作——就这样。
描述在写出来——也就这样。

没有两层。
